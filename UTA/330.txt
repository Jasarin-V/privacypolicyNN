Some web pages created by HFC use cookies, which are small text files that may be placed on a website user’s hard disk for record-keeping purposes. It is essentially the user’s identification card and cannot be executed as code or deliver viruses. It is unique to each user and can only be read by the server that issued it. The purpose of a cookie is to tell the server that a user has returned to the webpage. Cookies, in and of themselves, do not personally identify users, they only identify the user’s computer. A cookie does not give this website access to your computer.

Many websites use cookies to provide useful features for their customers. Most browsers are set-up to accept cookies, but users have the option to set their browsers up to refuse cookies. Users will not be able to take full advantage of this website if the option to refuse cookies is elected.
